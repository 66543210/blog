堆

堆可以看做是完全二叉树的数组表示

常见的堆的用法有：

  • 堆排序
  • 找出最大(最小)的k个数字
  • 让代码看起来更高牛效逼

堆分为：大根堆和小根堆

既然完全二叉树那么父节点，左子树节点和右子树节点在数组坐标上分别为i，i*2，i*2+1, 在c++或其他的int是整除的语言中，对于一个点j, 它的父节点为j / 2

大根堆

父节点的值都大于子节点的值

构建大根堆/插入新元素

给定一个数组构造大根堆：

  • 若只有一个点，那点的序列就是大根堆

  • 若数据>=2时就需要进行调整，从插入的那个小子树开始调整使其成为大根堆，然后再逐步向上调整小子树根节点所在的子树

  • 以[1, 2, 3, 4]为例子

    ①、插入1

    ②、插入2，[1, 2]

    可以看到在把2插入末尾时(此时为左子树)，不满足大根堆要求，因此进行调整也就是swap(root, max(left, right))之后变为

    [2, 1]

    ③、插入3，[2, 1, 3], 满足

    ④、插入4，[2, 1, 3, 4] 不满足

总结：在构建大根堆时需要层层调整，简单来说就是插入一个新元素时，若其大于根节点，则和根节点交换，交换之后需要基于这个新的根节点再递归向上遍历根节点，而这个过程也叫做向上调整

代码实现

void insert(vector<int>&list, int x) {
    list.push_back(x);
    shiftUp(list, list.size() - 1);
}
void shiftUp(vector<int>&list, int i) {
    int root = i / 2; // 整除
    if(root == i) {
        return; // 递归出口，最后的根节点
    }
    if(list[root] < list[i]) {
        swap(list[root], list[i]);
        shiftUp(list, root);
    }
}

原地创建

void createMaxHead(vector<int>& list) {
    if (list.size() <= 1) {
        return;
    }
    // 模拟空白数组逐个插入元素，那么原地构建相当于从头遍历每个元素作为插入的最后一个元素然后向上调整
    for (int i = 0; i < list.size(); i++) {
        shiftUp(list, i);
    }
}
取出元素

在大根堆中，根元素为最大的元素，当删除根元素后，结构上也需要进行调整以满足大根堆

以上例构建的大根堆为例

① 将最后一个元素补上根节点的位置

② 将这个新的根节点和较大的左右节点交换，即swap(root, max(left, right))

③交换往后，需要递归的调整构建以此节点为根节点的小子树，而这个过程称为向下调整

代码实现

int remove(vector<int>&list) {
    list.push_back(x);
    shiftUp(list, list.size() - 1);
}
void shiftDown(vector<int>&list, int i) {
    int left = i * 2;
    int right = i * 2 + 1;
    int max = i; // 初始时将当前点作为最大点
    if (left < list.size() && list[left] > list[max]) {
        max = left;
    }
    if (right < list.size() && list[right] > list[max]) {
        max = right;
    }
    if (max != i) {
        swap(list[i], list[max]);
        shiftDown(list, max);
    }
}

小根堆

小根堆就是父元素小于父子节点的树，在实现上和大根堆差不多，只是相关比较相反即可

构建小根堆/插入元素

和大根堆差不多也是需要层层调整，插入一个新元素时，若其小于根节点，则和根节点交换，交换之后需要基于这个新的根节点再递归向上遍历根节点，而这个过程也叫做向上调整

代码实现

void insert(vector<int>&list, int x) {
    list.push_back(x);
    shiftUp(list, list.size() - 1);
}
void shiftUp(vector<int>&list, int i) {
    int root = i / 2; // 整除
    if(root == i) {
        return; // 递归出口，最后的根节点
    }
    if(list[root] > list[i]) { // 和大根堆不同的地方
        swap(list[root], list[i]);
        shiftUp(list, root);
    }
}
