C++常用的算法库(algorithm)

算法篇

C++的算法库algorithm中封装了很多好用的方法，诸如查找最值元素、序列逆转等，在实际code的时候可以提供有力的支持

在使用前需要引入#include<algorithm.h>头文件

寻找最值

寻找容器内的最值

vector<int>v{0,1,2,3,4,1,2,-1};
cout<<*min_element(v.begin(),v.end())<<endl;//最小元素
cout<<*max_element(v.begin(),v.end())<<endl;//最大元素

使用自定义比较函数来寻找最值

class MyClass {
    int idc;
    int value;
}
bool mycompare (const MyClass& a, const MyClass& b) {
    return a.value > b.value;
}
// 传入自定义compare方法
min_element(v.begin(),v.end(), mycompare);
max_element(v.begin(),v.end(), mycompare);

// 使用自定义比较器后方法名前面的max和min

排序

sort
sort(beg, end); // 默认排序为升序(从小到大)
// 使用之定义比较器，一般适用于自定义类或结构体

bool mycompare (const MyClass& a, const MyClass& b) {
    return a.value - b.value;
}
sort(beg, end, mycompare);

// 简单的希望排序为降序时
vector<int>v{1,2,3,0,8};
bool descpare(int a, int b) {
    return a > b;
}
sort(v.begin(), v.end(), descpare);

去重

unqiue

在STL中unique函数是一个去重函数， unique的功能是去除相邻的重复元素>(只保留一个),其实它并不真正把重复的元素删除，是把重复的元素移到后面去了，方法返回去重后最后一个元素的地址

因此对于序列{1,1,3,1,5}使用后的序列为{1,2,1,5,5}

// 若要对全数组进行去重则可以先排序后删除
sort(v.begin(), v.end()); 
vector<string>::iterator end_unique =  unique(v.begin(), v.end()); 
v.erase(end_unique, v.end());

// 另外也可以使用set来回转一次去除重复数据

翻转

使用reverse可以对容器中的元素进行翻转

// 翻转string
string s = "abc";
reverse(s.begin(), s.end());
cout<<s; // ”cba“

// 翻转vector
vector<int>v{1,2,3,4,5};
reverse(v.begin(), v.end());
v; // {5,4,3,2,1}

交换

数组类的使用中有时会设置交换两个位置的值，可以使用swap

vector<int>v{1,2,3,4,5};
swap(v[1], v[2]);
// {1,3,2,4,5}

其他常用操作

string和number装换(stringstream)

需要包括头文件#include<sstream>

#include<sstream>

// 数字转string
stringstream ss;
string str;
ss<<123;
ss>>str;
cout<<str; // "123"

// string转数字
stringstream ss;
string str = "123"; // 若字符串是"123.3"这种浮点数字符串的话x应该为float，否则用int来接的话会变为123
int x;
ss<<str;
ss>>x;
cout<<x; // 123

使用文件进行输入输出

有时候输入用例的值很多，这个时候就使用文件作为输入流会方便很多

需要包括头文件#include<fstream>

// 使用外部文件作为cin输入
#include<iostream>
#include<fstream>
#include<vector>
using namespace std;
int main() {
    ifstream cin("aaa.txt"); // 和代码文件同目录
    int a, b, c, d;
    cin>>a>>b>>c>>d;
    cout<<"a="<<a<<",b="<<b<<",c="<<c<<",d="<<d<<endl;
    // 若想cout输出到文件则使用ofstream cout("aaa.txt"); 需要注意文件写入为覆盖非追加
    return 0;
}
